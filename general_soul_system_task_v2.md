# 105将魂系统接入任务书 v2

## 任务目标

将现有“将魂 / 武魂判辞”系统升级为 **105 将魂版本**。

新系统规则固定如下：

- 共 **15 个行为池**
- 每个行为池 **7 个等级**
- 共 **105 名将**
- **只输出 1 个主将魂**
- 不输出副将魂
- **等级优先按综分绝对区间判定**
- 其次才看行为特征与武将匹配
- 数据源以 `general_soul_system_105.csv` 为准，不允许 AI 自行重分配武将或重定等级

## 一、等级规则（硬规则，优先级最高）

请严格按下列综分区间定义 7 个等级：

- **Lv7**：综分 >= 400
- **Lv6**：350 <= 综分 <= 399
- **Lv5**：340 <= 综分 <= 349
- **Lv4**：330 <= 综分 <= 339
- **Lv3**：320 <= 综分 <= 329
- **Lv2**：310 <= 综分 <= 319
- **Lv1**：综分 <= 309

注意：

1. 等级不是玩法强弱，而是该武将在总榜中的综合段位。
2. 行为线内的武将必须先服从这个等级规则。
3. 即使某些武将和行为概念不完全贴合，也要优先服从综分档位。
4. 行为分再高，也不能把某个武将提升到不属于该 CSV 行里定义的等级。

## 五、推荐判定流程

### 第一步：计算 15 条行为分

根据玩家的本章战斗数据，分别计算 15 条行为线的分数。

### 第二步：取最高行为

分数最高的行为线，就是本局主行为线。

### 第三步：行为分归一化

将该行为线的表现映射成 0~100 的内部表现分。

### 第四步：映射到 Lv1~Lv7

建议使用统一阈值，例如：

- 0~14 => Lv1
- 15~29 => Lv2
- 30~44 => Lv3
- 45~59 => Lv4
- 60~74 => Lv5
- 75~89 => Lv6
- 90~100 => Lv7

### 第五步：按“行为 + Lv”在 CSV 中查表

例如：

- 行为 = 疾袭先登
- Lv = Lv6

查到武将 = 张辽

最终输出张辽作为本局主将魂。

## 七、建议评分模型（供 AI 直接实现）

建议每条行为线最终都输出 0~100 分。

### 通用做法

1. 先定义每条行为线的原始分公式
2. 再做归一化
3. 再 clamp 到 0~100

### 伪代码

```js
const score = clamp(Math.round(rawScore / expectedMax * 100), 0, 100);
```

### 说明

- `rawScore`：该行为线的实际表现值
- `expectedMax`：设计期望上限
- 不要求所有行为线完全同公式，但输出必须统一到 0~100

## 八、平局处理规则（必须实现）

### 第一优先：主标签命中数更多

- 疾袭先登优先看 快攻命中 / 打断 / 抢先手

例如：

- Boss 处决
- 精英处决

- 调息被打更少

固定优先级如下：

1. 疾袭先登
2. 破军重斩

## 九、等级映射与 CSV 查表规则（必须严格）

### 查表逻辑

系统不能“猜”武将，必须按下列规则执行：

1. 确定主行为线
2. 确定 Lv
3. 用 `(行为, Lv)` 作为唯一键，在 CSV 中查找
4. 必须只返回一条记录
5. 如果查不到，返回安全 fallback，不允许崩溃

### fallback 规则

若查表失败：

- 保留行为名称
- 等级默认降一级重查
- 最低降到 Lv1
- 如果仍失败，返回：
  - 武将 = “无名校尉”
  - 特技 = “—”
  - 其余数值为空或 0
  - 并在日志中标记查表失败

## 十、评价文案系统（建议实现）

不要为 105 名将手写 105 条独立评价。  
建议做成：

### 1. 行为模板

每个行为池有 1 套评价模板

### 2. 等级插值

同一行为池中：

- Lv1：写“已有雏形”
- Lv3：写“已见章法”
- Lv5：写“已有名将之势”
- Lv7：写“已臻上将气象”

### 示例

#### 疾袭先登

- Lv1：出手利落，已有先登抢势之形。
- Lv3：抢先压制颇见章法，已能在敌势未成前先下一城。
- Lv5：快攻凌厉、起手极快，已有名将先登之势。
- Lv7：疾而不乱，快而能稳，俨然赵子龙一流的先登之姿。

#### 破军重斩

- Lv1：重势初成，已懂得正面压制。
- Lv3：沉击破阵已有样子，不再只是蛮力挥砍。
- Lv5：重击、破防、收束一气呵成，已有大将斩阵气象。
- Lv7：沉击如山，一刀下去便要分高下。

实现上推荐：

- 每个行为池 4 条模板（Lv1/Lv3/Lv5/Lv7）
- Lv2 近似 Lv1
- Lv4 近似 Lv3
- Lv6 近似 Lv5
- Lv7 单独最强版本
